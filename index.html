📄 GAMERCORE MANAGER

Sobre el proyecto

GamerCore Manager es una aplicación web para entusiastas del hardware que permite armar configuraciones de PC, verificar compatibilidad entre componentes, obtener cotizaciones con impuestos y descuentos, y guardar el historial de builds en una base de datos centralizada.

Versión actual: 2.0.0 (Junio 2026) - Con soporte multiidioma y tema persistente.

🎯 Motivo de la app

Ayudar a usuarios sin experiencia a seleccionar componentes compatibles y a tener una referencia de precios y rendimiento estimado en juegos populares. También sirve como herramienta de aprendizaje.

🛠️ Tecnologías utilizadas

  • Frontend: HTML5, CSS3, JS puro
  • Backend: Node.js + Express
  • BD: MySQL (XAMPP)
  • Autenticación: bcrypt
  • PDF: jsPDF + jsPDF-AutoTable
  • Multiidioma: sistema propio basado en JSON

📖 Manual de uso

  1. Regístrate e inicia sesión
  2. Selecciona componentes
  3. Presiona "ARMAR PC" para ver análisis y guardar
  4. Usa "EXPORTAR A PDF" para generar lista de compras
  5. Revisa tu historial a la derecha
  6. Cambia el idioma y tema según tu preferencia
